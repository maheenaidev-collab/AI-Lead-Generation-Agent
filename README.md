# AI Lead Generation Agent

This project demonstrates an AI-powered lead generation system that can automatically analyze businesses and generate potential sales leads.

The AI agent takes a business niche and location as input and generates a list of potential leads with outreach messages.

## Features

- Business lead generation
- AI personalized outreach messages
- Lead data structure
- Automation-ready workflow

## Tech Stack

- Python
- OpenAI / Claude API
- FastAPI
- JSON data pipeline

## Example Input

Industry: Law Firms  
Location: New York

## Example Output

Lead 1:
Business: Smith & Partners Law Firm  
Email Message:
"Hi, I noticed your law firm specializes in personal injury..."

Lead 2:
Business: Johnson Legal Group  
Email Message:
"I help law firms automate their marketing systems..."
